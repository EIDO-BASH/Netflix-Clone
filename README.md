# Netflix-Inspired UI Clone

## Description
A front-end project inspired by Netflix’s interface, featuring login and sign-up pages with basic client-side validation and interactive behavior. Also includes a simple home page with a movie display.

## Features
- Login and sign-up page UI
- Form validation using JavaScript (e.g. password format with regex)
- User input checks with feedback for required fields
- Page redirection between authentication screens
- Home page with movie display using Bootstrap carousel
- Responsive layout across different screen sizes

## Tech Stack
- HTML
- CSS (Bootstrap)
- JavaScript
